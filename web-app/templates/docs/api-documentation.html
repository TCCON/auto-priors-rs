Authenticating

All the API endpoints for this site require that you authenticate your request. This requires that you have a token, which can be obtained from the token management page. For the rest of the documentation, we will assume that your token is saved to the file ~/.priors-api-key.

Important
Your tokens grant you access to the API, just like a username and password. Protect them just like you would your passwords: do not share them or send them between computer unencrypted (e.g., through email). If you need to copy a token from one computer to another, use something like scp or rsync (over SSH).
